# Step 1 — 성공기준
updated: 2026-05-14 (세션 13)

## 핵심 원칙
MVP부터 실제 사람에게 적용한다. 품질 기준 타협 없음.
"구조 검증용"이라는 이유로 아웃풋 품질을 낮추지 않는다.

## 성공 기준
- 아웃풋 4개 전부 생성됨 (비어있으면 실패)
- 각 아웃풋이 대상자 정보를 반영한 구체적 내용 (generic 금지)
- 실제 인플루언서가 읽고 자신의 이야기로 느낄 수 있는 수준
- 마크다운 리포트 파일 저장 완료
- 실제 인플루언서 1명에게 적용 + 전 과정 결과 기록 → 포트폴리오

## 최종 성공 기준 (실마케팅 검증)
- 구독자/팔로워 수 증가 (수치로 측정)
- CEO 자율 루프가 목표 달성까지 지속적으로 운영됨
  (핑구 개입 없이 CEO가 스스로 상태 파악 → 다음 액션 결정 → 에이전트 실행 반복)
- 성과 데이터는 핑구가 수동으로 CEO에게 제공 (MVP 기준)
  → 이후 자동 수집으로 확장 가능 (Phase 4+)

## 실패 기준
- 아웃풋 하나라도 비거나 generic
- 에이전트가 대상자 정보를 반영하지 못한 결과 출력
- 실제 적용 시 크리에이터가 공감하지 못함
- CEO가 목표를 인지하지 못하고 일회성으로 종료됨
