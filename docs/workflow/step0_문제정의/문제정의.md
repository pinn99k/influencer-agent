# Step 0 — 문제정의
updated: 2026-05-14 (세션 13)

## 문제 (한 문장)
인플루언서 육성에 필요한 대상 분석 → 시장 조사 → 플랫폼 선택 → 콘텐츠 기획을 1인이 수동으로 하면 시간·전문성 모두 부족해 확장이 안 된다.

## 해결
AI 에이전트 조직이 기획 전 과정을 자율 운영.
CEO 에이전트가 목표를 부여받으면 스스로 계획·실행·평가·루프를 돈다.

## 해결 안 하면
크리에이터마다 처음부터 수동 기획 → 확장 불가, 품질 들쭉날쭉.

## 목적
- 실제 인플루언서에게 적용 → 결과 기록 → 포트폴리오
- MVP 빠르게 검증, 이후 확장

## 범위 밖 (MVP — Spiral 0~2)
- 촬영·편집·업로드
- 웹 UI (Spiral 3에서 추가)
- 외부 API 연동 (Serper는 Spiral 1에서 포함)
- 부서 레이어 (Phase 4 이후)

## 제약
- 실제 사람에게 적용하는 결과물 → 품질 기준 높게 잡는다
