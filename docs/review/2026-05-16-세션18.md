## 배운 점

1. **Groq daily TPD 한도** — 하루 100,000 tokens. 집중 테스트 시 당일 소진 가능.
   → 캐싱 필수, 테스트 횟수 관리 필요

2. **8b vs 70b 포맷 준수** — llama-3.1-8b-instant는 출력 포맷을 자주 위반.
   → 프로덕션 에이전트는 70b 이상 필수

3. **경로 인코딩은 early fix** — 나중에 고치면 파일 경로 전수 수정 비용 큼.
   → 초기 설계에서 ASCII 경로 규칙 확립 필요 (ADR로 기록 필요)

4. **E2E 테스트 파일 독립성** — sys.path 조작 + os.chdir() + sys.stdout.reconfigure()
   → 세 줄이 빠지면 import 오류/인코딩 오류 발생
