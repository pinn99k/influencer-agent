# 세션 20 리뷰 — Spiral 3 Web UI 구현
날짜: 2026-05-19
Spiral: 3 (FastAPI + SSE Web UI)

---

## 목표
CLI 파이프라인을 브라우저에서 실시간 모니터링 + 회장 보고 응답 가능한 Web UI로 확장.

## 결정사항

### 1. EventEmitter 패턴
- CEO-SSE 디커플링: EventEmitter가 queue.Queue + FileManager.append_log() 브릿지
- CEO에 event_emitter=None 기본값 → CLI 모드 무영향
- 기존 101개 테스트 전량 통과 확인

### 2. threading.Event 회장 보고 게이트
- CEO 스레드 blocking + POST /decision으로 unblock
- 조건 9 (실행 단계 진입)는 정보성만 → blocking 안 함
- timeout=3600s로 좀비 스레드 방지

### 3. SessionManager 멀티잡
- dict[str, JobContext] + threading.Lock
- MAX_CONCURRENT_JOBS=5, 각 CEO 독립 FileManager

### 4. 보안
- CORS localhost 전용, Pydantic 입력 검증, DOMPurify XSS 방어
- 경로 탐색 방지: 화이트리스트 regex
