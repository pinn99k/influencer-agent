## 변경 목록 (7개)

### Change 1 — config.py SSoT (이전 세션 완료)

- 내용: 모든 경로(PROMPTS_DIR, KNOWLEDGE_DIR, OUTPUTS_DIR) + 상수(MAX_RETRY, LLM_*) 중앙 관리

### Change 2 — CEO 분해 (SRP)
- 기존: `agents/ceo.py` 355줄, 25개 메서드, 역할 4가지 혼재
- 신규:
  - `core/state_manager.py` — CEOStateManager: 상태 init/update/save
  - `core/report_builder.py` — ReportBuilder: briefing/rework/report/handover/plan 텍스트 조합
  - `core/prompt_loader.py` — PromptLoader: 프롬프트/지식 파일 로딩
  - `agents/ceo.py` — CEO: 오케스트레이션만 (에이전트 호출 순서/흐름)
- 하위 호환: `state` property + `_init_state`/`_update_state` 메서드 유지 (테스트 깨짐 방지)

### Change 3 — CompetitionAnalystAgent LSP fix
- 기존: `run()` override + `build_prompt(context, search_results=None)` (부모 서명 불일치)
- 수정: `run()` override 제거, Serper 호출을 `build_prompt(context)` 내부로 이동
- 효과: BaseAgent.run()이 호출 → build_prompt 안에서 serper 호출 → 서명 통일

### Change 4 — .gitignore 보안 (이전 세션 완료)
- .env, *.env, src/.env, src/outputs/ 추가

### Change 5 — FileManager path injection defense (이전 세션 완료)
- `_sanitize_name()` 추가, OUTPUTS_DIR 사용

### Change 6 — OutputValidator count_check regex fix
- 기존: `f"{i}." in output` → "2.5만 명" 같은 텍스트에서 false positive 발생
- 수정: `re.search(rf"^\s*{i}\.", output, re.MULTILINE)` — 줄 시작에서만 매칭

### Change 7 — BaseAgent context_key (DIP)
- `context_key: str = ""` class var 추가
- 각 에이전트: "대상_분석", "경쟁_분석", "플랫폼_추천", "컨셉_기획" 설정
- CEO에서 `AGENT_KEY_MAP` dict 완전 제거 → `agent.context_key`로 대체
- 효과: 에이전트 추가 시 AGENT_KEY_MAP 수동 동기화 불필요
