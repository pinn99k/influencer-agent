# ADR 003 — 모델 교체 기준 확정
date: 2026-05-12
status: 확정
step: 3 (MVP범위)

---

## 결정

- MVP 기본 모델: Groq llama3-70b (무료, 전 에이전트 동일)
- 교체 기준: 핑구가 아웃풋을 직접 읽고 "크리에이터에게 줄 수 없다" 판단 시
- 교체 단위: 전체 교체 아님 — NG 판정된 에이전트만 gpt-4o / Claude API로 개별 교체

## 이유

- 품질 판단 기준을 수치(metric)로 정의하면 MVP 단계에서 과잉 설계
- 실사람 적용이 목적이므로 "핑구가 줄 수 있는가"가 가장 직관적이고 정확한 기준
- 전체 교체는 불필요한 비용 증가 — 병목 에이전트만 교체해 비용 최소화

## 기각된 옵션

- 처음부터 gpt-4o 사용: 무료 Groq로 구조 검증 후 교체가 가성비 우선
- 수치 기반 품질 기준 (BLEU, 사용자 평점 등): MVP 범위 초과
- 에이전트별 모델 하드코딩: 관리 복잡도 증가, Phase 4+에서 API 할당 방식으로 대체 예정
