# ADR 004 — 경영 에이전트 API 할당 방식 → Phase 4+ 로드맵
date: 2026-05-12
status: 확정
step: 3 (MVP범위)

---

## 결정

경영 에이전트가 직무 에이전트에 필요 모델을 판단·할당하는 방식(옵션 C)은
MVP에서 제외하고 Phase 4+ 로드맵으로 확정한다.

## 이유

- MVP 목적은 에이전트 조직 구조 검증 — 모델 라우팅까지 포함하면 실패 지점 과다
- 경영 에이전트가 "어떤 에이전트에 어떤 모델이 적합한가"를 판단하는 로직 자체가
  별도 에이전트 수준의 설계 필요 → MVP 범위 초과
- Phase 3 안정화 후 실사용 데이터 기반으로 설계하는 것이 정확도 높음

## 기각된 옵션

- MVP에 포함: 실패 지점 증가, 디버깅 복잡도 과다
- 영구 제외: 확장성 측면에서 중요한 기능 → 로드맵 유지
