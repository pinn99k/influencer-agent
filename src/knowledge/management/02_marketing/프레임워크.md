# 마케팅관리 — CEO 판단 프레임워크
updated: 2026-05-16

## 사고흐름: 마케팅 관점 전략 판단

CEO가 초기 계획 수립 시 마케팅 관점에서 점검하는 흐름.

### Step 1. 콘텐츠 유형 매칭
```
질문: 이 사람의 전문성/성격에 가장 자연스러운 콘텐츠 유형은?
판단 방법:
  - 전문지식이 깊다 → 교육형 우선
  - 성격이 밝고 표현력 강하다 → 오락형 우선
  - 개인 경험이 공감 가능하다 → 공감형 우선
  - 소비/사용 경험이 풍부하다 → 리뷰형 우선
  - 복수 해당 시 → 1순위만 선택 (에이전트가 세부 조합)
출력: 1순위 콘텐츠 유형 + 근거
```

### Step 2. 타겟 오디언스 초기 가설
```
질문: 이 사람의 콘텐츠를 가장 필요로 하는 사람은 누구인가?
판단 방법:
  - 크리에이터의 직업/경험에서 "누가 이 사람에게 배우고 싶어할까?"
  - 크리에이터의 성격에서 "누가 이 사람과 시간을 보내고 싶어할까?"
  - 두 답의 교집합이 타겟
  - 교집합이 비면 → 타겟 불명확 (에이전트에게 탐색 지시)
출력: 타겟 페르소나 1줄 요약
```

### Step 3. 스윗스팟 평가
```
질문: 콘텐츠 스윗스팟이 존재하는가?
판단 방법:
  - 크리에이터 강점 + 타겟 니즈 교차점 확인
  - 교차점이 명확하면 → 전략 방향 확정 가능
  - 교차점이 모호하면 → 대상분석 에이전트에게 심층 탐색 지시
출력: 스윗스팟 존재 여부 + 1줄 설명
```

### Step 4. 채널 적합성 큰 그림
```
질문: 이 콘텐츠 유형이 어떤 플랫폼 군에서 강한가?
판단 방법:
  - 교육형 → 유튜브/블로그 (긴 콘텐츠 적합)
  - 오락형 → 틱톡/유튜브 숏폼 (짧고 강렬)
  - 공감형 → 인스타그램/블로그 (커뮤니티)
  - 리뷰형 → 유튜브/블로그 (신뢰+검색)
출력: 플랫폼 군 방향 (세부는 플랫폼추천 에이전트)
```

## 판단 원칙
- CEO는 마케팅 세부 전략을 세우지 않는다 — 큰 방향만 잡고 에이전트에게 위임
- 타겟이 불명확하면 억지로 정하지 않는다 — 에이전트에게 탐색 범위를 넓혀 지시
- 콘텐츠 유형은 1순위만 결정 — 복수 유형 조합은 컨셉기획 에이전트 영역
