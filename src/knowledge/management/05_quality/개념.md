# 품질관리 — 핵심 도메인 지식
updated: 2026-05-16

## 1. 산출물 품질 기준

### 품질의 3계층
| 계층 | 설명 | 검증 방법 |
|------|------|----------|
| 구조 품질 | 필수 섹션 존재, 형식 준수 | 자동 검증 (OutputValidator) |
| 내용 품질 | 구체성, 근거, 대상자 반영도 | CEO 주관 판단 |
| 실행 품질 | 크리에이터가 실제로 쓸 수 있는가 | 최종 판단 기준 |

자동 검증 통과는 최소 조건일 뿐이다. 내용 품질과 실행 품질은 CEO가 판단한다.

### Generic vs. Specific 구분법
| 판단 기준 | Generic (불합격) | Specific (합격) |
|----------|-----------------|----------------|
| 대상자 이름 | 없거나 형식적 | 맥락에 녹아 있음 |
| 조언 수준 | "꾸준히 하세요" | "주 2회 화요/금요 저녁 7시" |
| 근거 | 없음 | 대상자 정보에서 도출 |
| 경쟁자 언급 | 일반론 | 실명 + 구체적 비교 |
| 실행 가능성 | 추상적 방향 | 즉시 착수 가능한 액션 |

하나라도 Generic이면 rework 대상.

## 2. 실행 가능성 체크리스트

산출물이 다음 질문에 "예"로 답할 수 있어야 한다:

1. 크리에이터가 읽고 즉시 이해할 수 있는가?
2. 다음 행동이 명확한가? (무엇을, 언제, 어떻게)
3. 크리에이터의 현실적 제약(시간, 비용, 스킬) 안에서 가능한가?
4. 대상자 고유 정보가 빠지면 성립하지 않는 내용인가?

4번이 핵심이다. 대상자 이름만 바꿔도 성립하는 산출물은 가치가 없다.

## 3. 에이전트별 품질 초점

| 에이전트 | 핵심 품질 기준 |
|---------|--------------|
| 대상분석 | 강점/약점이 대상자 고유 정보에서 도출되었는가 |
| 경쟁분석 | 실제 경쟁자가 실명으로 언급되고, 차별점이 구체적인가 |
| 플랫폼추천 | 추천 이유가 대상자 특성과 연결되는가 |
| 컨셉기획 | 콘텐츠 아이디어가 즉시 제작 가능한 수준인가 |
