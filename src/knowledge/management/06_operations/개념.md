# 운영관리 — 핵심 도메인 지식
updated: 2026-05-16

## 1. 콘텐츠 생산 사이클

### 표준 워크플로우
```
기획 → 제작 → 편집 → 업로드 → 분석 → 개선
```

각 단계의 소요 시간은 콘텐츠 유형에 따라 크게 다르다:

| 콘텐츠 유형 | 기획 | 제작 | 편집 | 전체 사이클 |
|------------|------|------|------|------------|
| 숏폼 (60초) | 30분 | 1시간 | 30분 | 2~3시간 |
| 브이로그 | 1시간 | 촬영일 | 3~5시간 | 1~2일 |
| 교육 영상 (10분) | 2~3시간 | 2시간 | 3~5시간 | 1~2일 |
| 블로그 글 | 1~2시간 | 2~4시간 | 30분 | 반나절 |
| 풀 프로덕션 | 1일+ | 1일+ | 1일+ | 3일~1주 |

### 생산성 함정
- "매일 업로드" = 품질 저하 or 번아웃
- "완벽주의" = 업로드 빈도 0
- 적정 빈도는 크리에이터의 가용 시간과 콘텐츠 유형으로 결정

## 2. 리소스 배분

### 크리에이터 가용 리소스
| 리소스 | 설명 | CEO 판단 포인트 |
|--------|------|----------------|
| 시간 | 본업 외 투입 가능 시간 | 주당 가용 시간 파악 |
| 장비 | 카메라, 마이크, 편집 SW | 진입 장벽 판단 |
| 스킬 | 촬영, 편집, 글쓰기 능력 | 초기 학습 곡선 예측 |
| 비용 | 초기 투자 + 월 운영비 | 지속 가능성 판단 |

### 80/20 원칙
- 전체 성과의 80%는 20%의 콘텐츠에서 나온다
- 초기에는 양보다 "히트 가능성 높은 콘텐츠"에 리소스 집중
- 나머지 80% 콘텐츠는 일관성 유지용

## 3. 일정 관리

### 콘텐츠 캘린더 기본 구조
- **주간 단위**: 업로드 요일/시간 고정 (알고리즘 + 습관 형성)
- **월간 단위**: 테마/시리즈 기획 (일관성 + 다양성 균형)
- **분기 단위**: 성장 지표 리뷰 + 전략 수정

### 지속 가능한 빈도 공식
```
적정 빈도 = 가용 시간 / 콘텐츠 사이클 시간 x 0.7
```
0.7은 예비 시간(컨디션, 돌발 상황). 이 공식 초과 시 번아웃 리스크.
