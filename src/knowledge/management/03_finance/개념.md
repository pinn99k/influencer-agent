## 1. API 비용 구조

### 현재 시스템 비용 요소
| 항목 | 단가 | 월 예상 | 비고 |
|------|------|---------|------|
| Groq llama3-70b | 무료 (rate limit 30 req/min) | 0원 | 속도 제한이 실질 비용 |
| Serper API | 무료 2,500건/월 | 0원 | 초과 시 유료 전환 필요 |
| 스토리지 | 마크다운 파일 | 0원 | 로컬 저장 |

### 비용 확장 시나리오
| 시나리오 | 추가 비용 | 트리거 조건 |
|---------|----------|------------|
| 모델 업그레이드 (GPT-4o) | 월 20~50달러 | 품질 NG 반복 시 |
| Serper 유료 전환 | 월 50달러 (50,000건) | 경쟁분석 대량 실행 시 |
| 클라우드 배포 | 월 10~30달러 | Spiral 3 웹 UI 시 |

### 비용 감도
현재 MVP는 비용 0원 구조. 어떤 변경이든 비용 발생은 회장 보고 대상 (조건 2).

## 2. ROI 계산

### 인플루언서 에이전트 ROI 공식
```
ROI = (크리에이터 성장 가치 - 시스템 운영 비용) / 시스템 운영 비용
```

### 성장 가치 측정 지표
| 지표 | 측정 방법 | 가치 판단 |
|------|----------|----------|
| 구독자 증가율 | 월간 순증 | 가장 직접적 성공 지표 |
| 콘텐츠 참여율 | 좋아요/댓글/공유 비율 | 오디언스 품질 지표 |
| 수익화 진척 | 첫 수익 발생 여부 | 장기 가치 지표 |

### MVP 단계 ROI 판단
MVP에서는 비용 0원이므로 ROI 공식보다 "투입 시간 대비 산출물 품질"이 핵심.
- 에이전트 1회 실행에 5분 이내 → 효율적
- 산출물이 수동 작업 2시간 이상 대체 → 가치 있음
- 산출물 품질이 "핑구가 직접 전달 가능" 수준 → ROI 성립

## 3. 투자 대비 효과 기준

### MVP 투자 판단 원칙
- 무료 도구 범위 내에서 최대 품질 추구
- 유료 전환은 무료 한계가 명확히 드러난 후에만
- "혹시 모르니까 유료로" 금지 — 근거 없는 선제 투자는 낭비
