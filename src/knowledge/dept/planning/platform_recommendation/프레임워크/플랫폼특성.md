# 인플루언서 도메인 — 03 플랫폼 특성
# 목적: 플랫폼추천 에이전트가 플랫폼별 특성을 기반으로 추천할 때 참조
updated: 2026-05-14

---

## 유튜브 (YouTube)

### 알고리즘 특성
- **발견**: 검색 + 추천 피드. 검색 기반 유입이 강함 → 키워드 중요.
- **수명**: 동영상 수명 길다. 1년 전 영상도 검색으로 계속 유입.
- **성장 속도**: 느림. 초기 구독자 1,000명까지 평균 6~18개월.

### 유리한 크리에이터 유형
- 정보·튜토리얼·How-to 콘텐츠
- 롱폼 스토리텔링 (10분 이상)
- 전문 지식 기반 심층 설명
- 꾸준한 편집 능력 필요

### 수익화 시점
- 구독자 1,000명 + 시청 시간 4,000시간 → 유튜브 파트너십
- 한국 평균: 구독자 1만 명 이상부터 협찬 가능성 시작

### 주의
- 편집 비용·시간이 가장 높다.
- 초기 성과가 없어도 6개월은 유지해야 데이터 쌓임.

## 인스타그램 (Instagram)

### 알고리즘 특성
- **발견**: 릴스(Reels) 중심. 팔로워 없어도 릴스로 바이럴 가능.
- **수명**: 짧다. 일반 피드 포스트는 24~48시간. 릴스는 길게 유통.
- **성장 속도**: 릴스 바이럴 시 빠름. 단, 변동성 높음.

### 유리한 크리에이터 유형
- 비주얼 강점 (외모, 제품, 공간, 음식)
- 짧은 영상으로 임팩트 전달 가능한 콘텐츠
- 라이프스타일, 패션, 뷰티, 인테리어
- 팬과의 소통·친밀감 중심

### 수익화 시점
- 팔로워 1만 명 이상 → 링크 추가, 협찬 가능
- 팔로워 5만 명 이상 → 인플루언서 마케팅 본격화

### 주의
- 알고리즘 변화가 잦다. 릴스 의존도 높으면 변화에 취약.
- 팔로워 수보다 인게이지먼트율(댓글·저장)이 협찬 단가에 더 영향.

## 틱톡 (TikTok)

### 알고리즘 특성
- **발견**: 팔로워 0명도 바이럴 가능. 완전한 발견형 플랫폼.
- **수명**: 매우 짧다. 48~72시간이 핵심.
- **성장 속도**: 가장 빠름. 단, 팬 충성도 낮은 경향.

### 유리한 크리에이터 유형
- 숏폼 임팩트 (15~60초 안에 핵심)
- 트렌드 반응 속도 빠른 크리에이터
- 10~20대 타겟
- 유머·챌린지·반전 형식

### 수익화 시점
- 한국에서 직접 수익화 구조 약함 (크리에이터 펀드 미지원).
- 협찬·유튜브·인스타 유입 채널로 활용하는 것이 현실적.

### 주의
- 20~30대 이상 타겟이면 효과 낮음.
- 콘텐츠 수명이 짧아 지속 업로드 부담이 크다.

## 플랫폼 선택 판단 기준

| 조건 | 추천 플랫폼 |
|------|------------|
| 전문 지식 + 롱폼 선호 | 유튜브 |
| 비주얼 강점 + 라이프스타일 | 인스타그램 |
| 10~20대 타겟 + 트렌드 반응 빠름 | 틱톡 |
| 편집 시간 제약 큼 | 인스타그램 릴스 (상대적으로 짧음) |
| 초기 팔로워 없음 + 빠른 성장 원함 | 틱톡 or 인스타 릴스 |
| 장기적 자산으로 콘텐츠 축적 원함 | 유튜브 |
