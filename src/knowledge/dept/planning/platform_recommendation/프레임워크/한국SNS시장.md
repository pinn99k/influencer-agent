## 핵심 원칙

한국 SNS 시장은 글로벌 트렌드와 다른 특이점이 있다.
플랫폼 추천·경쟁 분석·컨셉 기획 시 이 특이점을 반영한다.

## 한국 시장 플랫폼 지형

### 글로벌 플랫폼 사용 현황 (2024~2025 기준)
| 플랫폼 | 주 사용 연령 | 특이점 |
|--------|------------|--------|
| 유튜브 | 전 연령 (특히 30~50대 강함) | 국내 OTT보다 시청 시간 길다. 정보 검색도 유튜브 먼저. |
| 인스타그램 | 20~30대 중심 | 팔로잉 기반 소비. 릴스보다 피드 저장 문화 강함. |
| 틱톡 | 10~20대 중심 | 성인 크리에이터 수익화 구조 약함. |
| 카카오 | 전 연령 | SNS보다 커뮤니케이션 채널. 채널(카카오톡 채널)로 팬 관리 가능. |
| 네이버 블로그 | 30~50대 | 정보 검색 + 리뷰. 협찬 콘텐츠 채널로 여전히 유효. |
| 네이버 인플루언서 | 20~40대 | 검색 상단 노출 + 광고 수익. 블로그 기반. |

## 한국 시장 특이점

### 1. 유튜브가 검색 엔진 역할
"어떻게 하는지" 궁금하면 네이버 대신 유튜브 검색하는 문화.
→ 튜토리얼·리뷰·일상 정보 콘텐츠는 유튜브 SEO가 중요.

### 2. 네이버 블로그 + 인플루언서 프로그램 생태계
블로그 체험단·협찬 문화 강함. 소규모 크리에이터도 협찬 가능.
→ 뷰티·맛집·여행·육아 분야에서 특히 유효.

### 3. 카카오톡 채널을 팬 관리 채널로 활용
대형 크리에이터들이 인스타·유튜브에서 유입 → 카카오 채널로 락인.
→ 팬덤 관리·유료 멤버십 확장 경로.

### 4. 인스타그램 저장 문화
한국 인스타그램 사용자는 "나중에 참고"를 위해 저장 많이 함.
→ 저장하고 싶어지는 정보형 콘텐츠(카드뉴스, 인포그래픽) 효과적.

### 5. 협찬 단가 기준
- 인스타그램: 팔로워 1만 명당 협찬 단가 10~30만 원 수준 (분야별 상이)
- 유튜브: 구독자보다 평균 조회수 기준. 1만 뷰당 단가 형성.
- 틱톡: 한국에서 협찬 단가 낮음. 유입 채널로 활용.

## 분야별 시장 포화도 (2024~2025 기준)

| 분야 | 포화도 | 진입 전략 |
|------|--------|----------|
| 먹방·요리 | 매우 높음 | 니치화 필수 (특정 식재료, 지역 음식 등) |
| 뷰티·헤어 | 높음 | 전문가 시각 또는 특정 타겟(40대 뷰티 등) |
| 피트니스·운동 | 높음 | 특정 운동 종목 또는 특정 신체 고민 |
| 직업 브이로그 | 중간 | 희소 직업일수록 유리 |
| 육아 | 중간 | 특정 상황(다둥이, 워킹맘 등) 특화 |
| 재테크·부업 | 높음 | 실증 데이터 중심으로 차별화 |
| 로컬·지역 콘텐츠 | 낮음 | 지역 기반 팬덤 형성 가능 |

## 에이전트 활용 방법

- **경쟁분석**: 한국 시장 포화도 + 플랫폼 지형 반영해서 공백 분석
- **플랫폼추천**: 타겟 연령대 + 분야 + 한국 사용자 분포 교차 판단
- **컨셉기획**: 저장하고 싶은 콘텐츠 / 검색에 걸리는 콘텐츠 형식 우선
