# 인플루언서 도메인 — 02 경쟁 분석
# 목적: 경쟁분석 에이전트가 시장 포지션과 공백을 찾을 때 참조
updated: 2026-05-14

---

## 핵심 원칙

경쟁 분석의 목적은 "이미 있는 것"을 파악해서 "없는 것"을 찾는 것이다.
유사 크리에이터를 나열하는 것이 목적이 아니다.

---

## 포지션 분석 방법

### 1. 포지션 축 설정
크리에이터 시장은 아래 축으로 포지션을 나눈다.

| 축 | 양 끝점 |
|----|---------|
| 콘텐츠 깊이 | 입문·대중 ↔ 전문·심층 |
| 톤 | 밝고 가벼움 ↔ 진지하고 무거움 |
| 타겟 연령 | 10~20대 ↔ 30~40대 이상 |
| 형식 | 숏폼(1분 이하) ↔ 롱폼(10분 이상) |
| 접근 방식 | 감성·스토리 ↔ 정보·분석 |

### 2. 유사 포지션 크리에이터 파악
Serper 검색 결과를 기반으로 유사 크리에이터 3개 이상을 찾는다.

각 크리에이터에 대해:
- 포지션 축 상 위치
- 주요 콘텐츠 형식
- 구독자 규모 (있으면)
- 강점으로 보이는 점

### 3. 공백(Gap) 발견
유사 포지션 크리에이터들을 지도에 배치했을 때 비어 있는 좌표가 공백이다.

공백 발견 질문:
- "이 분야에서 {특정 톤}으로 {특정 타겟}을 겨냥하는 크리에이터가 있는가?"
- "전문 지식을 {입문자 언어}로 설명하는 크리에이터가 있는가?"
- "실무자 시각을 {소비자 관점}으로 번역하는 콘텐츠가 있는가?"

---

## 차별 전략 도출

공백을 발견한 후, 대상자가 그 공백을 채울 수 있는지 판단한다.

판단 기준:
- 대상자의 강점과 공백이 겹치는가? (대상분석 결과 연결)
- 대상자가 그 포지션을 유지할 수 있는가? (지속 가능성)
- 해당 포지션에 수요(타겟 관심)가 있는가?

---

## 검색 키워드 생성 방법

Serper에 넣을 쿼리는 아래 조합으로 만든다.

```
{직업/분야} + {플랫폼} + 크리에이터
{직업/분야} + 유튜버 / 인스타그래머
{특기} + SNS + 채널
```

예시 (미용사):
- "미용사 유튜버"
- "헤어 전문가 인스타"
- "살롱 직원 브이로그"

---

## 분석 시 금지 사항

- 특정 실존 인물 계정 명시 (채널 유형·포지션만 언급)
- Serper 결과 없이 경쟁사 단정 (→ [추정] 태그)
- 대상자 플랫폼 추천 (→ 플랫폼추천 에이전트 영역)
- 공백 없다고 단정 (→ 항상 상대적 공백 존재)
