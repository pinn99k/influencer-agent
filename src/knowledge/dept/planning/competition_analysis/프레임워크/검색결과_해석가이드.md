# 검색 결과 해석 가이드
# 목적: 경쟁분석 에이전트가 Serper 검색 결과에서 크리에이터 정보를 정확히 추출할 때 참조
updated: 2026-05-21

---

## Serper 검색 결과 구조

Serper API는 Google 검색 결과를 JSON으로 반환한다.
경쟁분석에서 주로 활용하는 필드:

```
organic[]:
  title:   검색 결과 제목
  snippet: 검색 결과 요약 텍스트 (2~3문장)
  link:    URL
```

## 유튜브 채널 정보 추출법

### title에서 채널명 추출

패턴 1: "영상 제목 - 채널명 - YouTube"
- 추출: 마지막 " - YouTube" 제거 후, 그 앞의 " - " 뒤가 채널명
- 예: "펌 하기 전에 꼭 보세요 - 헤어짱 - YouTube" → 채널명: "헤어짱"

패턴 2: "채널명 - YouTube"
- 채널 메인 페이지가 결과에 나온 경우
- 예: "헤어짱 - YouTube" → 채널명: "헤어짱"

패턴 3: "영상 제목 | 채널명"
- 일부 결과에서 파이프(|)로 구분
- 예: "2026 여름 트렌드 컬러 | 살롱비비" → 채널명: "살롱비비"

패턴 4: title에 채널명 없음
- snippet이나 link에서 추출 시도
- link 패턴: youtube.com/@채널명 또는 youtube.com/c/채널명

### snippet에서 구독자 수 추출

한국어 구독자 수 표기 패턴:
| 원문 | 파싱 결과 |
|------|----------|
| "구독자 2.13만명" | 21,300 |
| "구독자 2.13만" | 21,300 |
| "구독자 213명" | 213 |
| "구독자 1.5천명" | 1,500 |
| "구독자 15만" | 150,000 |
| "구독자 1만명" | 10,000 |
| "구독자 100만" | 1,000,000 |

영어 표기 패턴:
| 원문 | 파싱 결과 |
|------|----------|
| "2.13K subscribers" | 2,130 |
| "21.3K subscribers" | 21,300 |
| "1.5M subscribers" | 1,500,000 |
| "213 subscribers" | 213 |

### 숫자 파싱 규칙

한국어 단위 변환:
- "만" = x 10,000
- "천" = x 1,000
- "억" = x 100,000,000 (크리에이터에선 거의 없음)

소수점 처리:
- "2.13만" = 2.13 x 10,000 = 21,300
- "1.5천" = 1.5 x 1,000 = 1,500
- "3.7만" = 3.7 x 10,000 = 37,000

영어 단위 변환:
- "K" = x 1,000
- "M" = x 1,000,000
- "B" = x 1,000,000,000

## 크리에이터 vs 비크리에이터 구분법

검색 결과에는 크리에이터가 아닌 결과도 섞여 나온다.

### 크리에이터로 판단하는 신호
- URL이 youtube.com/@, youtube.com/c/, youtube.com/channel/
- URL이 instagram.com/{username}
- snippet에 "구독자", "팔로워", "채널", "크리에이터" 포함
- title에 개인 이름 또는 닉네임 느낌의 단어

### 비크리에이터 (제외 대상)
- 뉴스 기사: URL이 news., article., blog. 등 미디어 도메인
- 플랫폼 공식: URL이 youtube.com/results, youtube.com/feed 등
- 교육 기관: "미용학원", "아카데미", "뷰티스쿨" 등
- 커뮤스/포럼: "커뮤니티", "카페", "지식인" 등
- 제품/브랜드: snippet에 "제품", "가격", "구매" 중심

### 기업 채널 vs 개인 크리에이터
- 기업: 채널명에 "TV", "공식", "Official" 포함, 또는 브랜드명
- 개인: 닉네임, 이름, 별명 형태
- 경쟁분석에서는 개인 크리에이터만 대상 (기업 채널은 참고만)

## 검색 쿼리 최적화

### 기본 쿼리 구조
```
{직업/분야} + {크리에이터 식별 키워드}
```

### 크리에이터 식별 키워드 (한국어)
- "유튜버", "유튜브", "채널"
- "크리에이터", "인플루언서"
- "인스타", "인스타그램", "릴스"
- "틱톡", "틱토커"
- "브이로그", "일상"

### 효과적인 쿼리 예시 (미용사 기준)
| 쿼리 | 기대 결과 |
|------|----------|
| "미용사 유튜버" | 미용사 출신 유튜브 채널 |
| "헤어 크리에이터 OR 헤어 유튜버" | 헤어 전문 크리에이터 |
| "미용실 브이로그 OR 살롱 브이로그" | 미용실 일상 콘텐츠 |
| "헤어컬러 전문 채널" | 컬러 특화 크리에이터 |
| "셀프 헤어 스타일링 유튜브" | 셀프 케어 교육 채널 |
| "미용사 인스타 릴스" | 인스타 활동 미용사 |

### 검색 결과 부족할 때 대안 전략

1단계: 키워드 확장
- "미용사" → "헤어 전문가", "헤어 디자이너", "스타일리스트"
- "유튜버" → "채널", "크리에이터", "인플루언서"

2단계: 플랫폼 특화 검색
- "site:youtube.com 미용사" → 유튜브 내 결과만
- "site:instagram.com 헤어디자이너" → 인스타 내 결과만

3단계: 간접 키워드
- "헤어 변신 쇼츠" → 콘텐츠 유형으로 검색
- "before after 헤어 컬러" → 콘텐츠 포맷으로 검색

4단계: 영어 검색 (한국 결과 부족 시)
- "hairstylist youtuber korea"
- "korean hair transformation"

## 검색 결과에서 포지셔닝 정보 추출

### snippet 분석 포인트
1. 콘텐츠 유형: "튜토리얼", "리뷰", "브이로그", "변신" 등 키워드
2. 타겟: "초보자", "전문가", "20대", "남성" 등
3. 톤: "재밌는", "전문적인", "차분한" 등
4. 업로드 빈도: "매주", "매일", "주 3회" 등 (있으면)

### 정보 부족 시 처리
- 구독자 수 미확인: "[추정] 구독자 약 N명" 또는 "구독자 정보 없음"
- 채널명 미확인: "관련 채널 발견되지 않음"으로 명시
- 절대 추측으로 구독자 수를 만들어내지 않음

## 검색 결과 신뢰도 판단

| 신호 | 신뢰도 | 처리 |
|------|--------|------|
| youtube.com URL + 구독자 수 표기 | 높음 | 그대로 사용 |
| 뉴스/블로그에서 언급 + 구독자 수 | 중간 | 시점 확인 (오래된 데이터 주의) |
| snippet에 구독자 수 없음 | 낮음 | [추정] 또는 "미확인" 표기 |
| 구독자 수 시점 불명 | 중간 | "검색 시점 기준" 표기 |

## 에이전트 활용 가이드

이 파일을 참조할 때 에이전트는:
1. Serper 결과를 받으면 먼저 크리에이터 vs 비크리에이터 필터링
2. 채널명과 구독자 수를 위 패턴으로 추출
3. 추출 불가 시 [추정] 또는 "미확인"으로 명시 (절대 만들어내지 않음)
4. 포지셔닝 정보는 snippet에서 간접 추출
5. 검색 결과가 3개 미만이면 대안 전략으로 쿼리 확장 시도
