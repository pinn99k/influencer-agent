# 인플루언서 도메인 — 05 컨셉 기획
# 목적: 컨셉기획 에이전트가 콘텐츠 컨셉과 영상 아이디어를 도출할 때 참조
updated: 2026-05-14

---

## 핵심 원칙

컨셉은 "무엇을 만들 것인가"가 아니라 "누구에게, 무엇을, 왜 봐야 하는가"다.
컨셉이 없으면 콘텐츠는 방향 없이 쌓인다.

---

## 좋은 컨셉의 3가지 조건

1. **반복 가능**: 이 컨셉으로 100개의 영상을 만들 수 있는가?
2. **차별 가능**: 이미 많은 크리에이터가 하고 있지 않은가?
3. **타겟 공명**: 타겟이 "이건 나를 위한 채널이다"라고 느끼는가?

---

## 컨셉 도출 방법

### 1. 차별점 × 타겟 매트릭스
대상 분석의 차별점과 타겟 연령대를 교차한다.

```
차별점: {대상자만 줄 수 있는 것}
타겟: {가장 원하는 연령·상황}
교차점: {차별점이 타겟의 어떤 니즈를 해결하는가}
```

### 2. 포지셔닝 틈새 채우기
경쟁 분석에서 발견한 공백(Gap)을 컨셉으로 전환한다.

```
공백: {경쟁분석 결과}
컨셉화: "이 채널은 {공백을 채우는 방식}을 하는 채널"
```

### 3. 컨셉 3개 도출
서로 다른 방향으로 3개를 도출한다. 유사한 3개는 의미 없다.

컨셉 다양화 축:
- 톤: 전문적 ↔ 친근한
- 형식: 정보형 ↔ 스토리형
- 깊이: 입문 ↔ 심층

---

## 컨셉 작성 형식

```
#### 컨셉 A: {컨셉 제목}
한 줄 정의: {이 채널은 ___을 하는 채널이다}
타겟: {누구를 위한 채널인가}
차별점 활용: {대상자의 어떤 강점이 이 컨셉을 가능하게 하는가}
성격 반영: {대상자 성격이 어떤 방식으로 녹아드는가}
공백 연결: {경쟁분석의 어떤 공백을 채우는가}
예상 콘텐츠 형식: {숏폼/롱폼, 정보형/스토리형 등}
```

## 영상 아이디어 도출 방법

컨셉 확정 후 각 컨셉에서 실제 제작 가능한 영상 아이디어를 뽑는다.

### 좋은 아이디어의 조건
- **제목이 바로 나오는 것**: "00하는 사람은 꼭 봐야 하는 것"
- **내부자 시각**: 일반인이 모르는 업계 상식
- **대비와 반전**: "당신이 알고 있는 것 vs 실제"
- **검색 가능**: 유튜브에서 찾을 법한 키워드 포함

### 아이디어 5개 작성 형식
```
1. {영상 제목 (실제 올릴 수 있는 수준)}
   - 컨셉: {어떤 컨셉과 연결}
   - 핵심 훅: {첫 5초에 말할 내용}
   - 타겟이 보는 이유: {왜 클릭하는가}
```

## 니치 vs 대중 판단 기준

| 조건 | 니치 공략 | 대중 공략 |
|------|----------|----------|
| 분야 포화도 | 높음 | 낮음 |
| 크리에이터 전문성 | 매우 높음 | 중간 |
| 타겟 규모 | 작지만 충성 | 크고 경쟁 강함 |
| 성장 속도 | 느리지만 안정 | 빠르지만 변동 |
| 초기 전략 | 니치 우선 → 확장 | 대중은 MVP에서 리스크 큼 |

MVP에서는 니치 공략 우선 권장.
이유: 초기 팬덤이 생기면 대중 확장이 가능하지만, 대중 실패 시 방향 전환이 어렵다.

---

## 분석 시 금지 사항

- 컨셉 3개가 방향이 비슷한 경우 (다시 도출)
- 영상 아이디어가 제목만 있고 훅·타겟 이유 없는 경우
- 대상자 성격·강점이 컨셉에 반영되지 않은 Generic 컨셉
- "꾸준히 올리세요" 같은 실행 조언 (이 에이전트 역할 아님)
