# 숏폼 바이럴 공식
# 목적: 컨셉기획 에이전트가 영상 아이디어와 콘텐츠 구조를 설계할 때 참조
updated: 2026-05-21

---

## 핵심 원칙

숏폼의 성패는 첫 1~3초에 결정된다. "훅(Hook)"이 없으면 스크롤된다.
숏폼 = 완벽한 영상이 아니라 "멈추게 하는" 영상이다.

---

## 훅(Hook) 공식: 첫 1~3초

### 공식 1: 질문형 훅
- 패턴: "OO 아직도 이렇게 하세요?"
- 작동 원리: 시청자의 궁금증 유발 → 답을 듣기 위해 시청 지속
- 예시:
  - "드라이 할 때 아직도 이렇게 하세요?"
  - "펌 하고 이거 안 하면 100% 망합니다"
  - "미용사가 진짜 싫어하는 주문 3가지 아세요?"

### 공식 2: 충격/반전형 훅
- 패턴: 예상 밖의 비주얼로 시작
- 작동 원리: "이게 뭐지?" → 알아내기 위해 시청
- 예시:
  - 손상 심한 머리 클로즈업으로 시작 → 복구 과정
  - 탈색 5번 된 머리 → 완벽한 컬러 변신
  - 고객 요청 사진(극단적) → 실제 시술 결과

### 공식 3: 비교/대비형 훅
- 패턴: 화면 분할 또는 전후 대비
- 작동 원리: 차이가 클수록 시선 고정
- 예시:
  - 좌: 손상모 / 우: 시술 후 (동시 비교)
  - "살롱 10만원 vs 집에서 1만원" (가격 대비)
  - "인터넷 방법 vs 미용사 방법" (전문가 대비)

### 공식 4: 비포/에프터형 훅
- 패턴: "전" 상태로 시작 → 결과는 마지막에
- 작동 원리: 결과 궁금증 → 끝까지 시청
- 예시:
  - 뿔뿔 곱슬머리 → 매끈한 매직 결과
  - 칙칙한 블랙 → 트렌드 컬러 변신
  - 얼굴형과 안 맞는 컷 → 이미지 체인지

### 공식 5: 리스트형 훅
- 패턴: "OO 3가지", "절대 하면 안 되는 OO"
- 작동 원리: 숫자가 구체성 제공 → "나도 해당되나?" 확인 심리
- 예시:
  - "미용사가 절대 추천 안 하는 홈케어 3가지"
  - "펌 오래 유지하는 방법 5가지"
  - "미용실에서 돈 버리는 행동 TOP 3"

## 뷰티 숏폼 성공 구조

### 기본 구조 (30~60초)

```
[0~3초] 훅 -- 시선 고정
[3~10초] 문제 제시 -- 공감 유도
[10~25초] 과정/해결 -- 핵심 콘텐츠
[25~28초] 결과 -- 시각적 만족
[28~30초] CTA -- 행동 유도
```

### CTA (Call to Action) 패턴
- 구독 유도: "더 많은 꿀팁은 팔로우"
- 댓글 유도: "여러분은 어떤 스타일 좋아하세요?"
- 저장 유도: "나중에 미용실 갈 때 보여주세요"
- 공유 유도: "이거 몰랐으면 친구한테도 알려주세요"

### 뷰티 특화 구조 변형

변형 A: 변신형 (가장 바이럴 확률 높음)
```
[0~2초] 시술 전 상태 (충격적일수록 좋음)
[2~20초] 시술 과정 (타임랩스/하이라이트)
[20~28초] 결과 공개 (슬로모션 + 각도 변경)
[28~30초] 리액션 또는 CTA
```

변형 B: 교육형 (검색 유입 강점)
```
[0~3초] 질문/문제 제시
[3~25초] 단계별 설명 (자막 필수)
[25~30초] 핵심 정리 + CTA
```

변형 C: 실패/수정형 (신뢰도 구축)
```
[0~3초] 실패 상황 공개
[3~10초] 왜 실패했는지 설명
[10~25초] 수정/복구 과정
[25~30초] 최종 결과 + 교훈
```

## 조회수 높은 뷰티 숏폼 제목 패턴 (검증됨)

### 패턴별 예시

| 패턴 | 예시 | 왜 작동하는가 |
|------|------|-------------|
| "OO하면 안 되는 이유" | "펌 후 빗으면 안 되는 이유" | 경고 = 클릭 욕구 |
| "OO vs OO" | "살롱 케어 vs 홈케어" | 비교 = 결과 궁금증 |
| "미용사가 알려주는 OO" | "미용사가 알려주는 드라이 꿀팁" | 전문가 = 신뢰 |
| "이것만 하면 OO" | "이것만 하면 펌 2배 오래감" | 간단한 해결책 약속 |
| "절대 OO하지 마세요" | "절대 이 샴푸 쓰지 마세요" | 금지 = 호기심 |
| "OO 전에 꼭 보세요" | "염색 전에 꼭 보세요" | 실용적 긴급성 |
| "OO만원으로 OO" | "1만원으로 살롱급 트리트먼트" | 가성비 = 관심 |
| "처음 공개" | "수습 미용사의 첫 시술 처음 공개" | 독점 = 희소성 |

## 해시태그 전략

### 해시태그 3층 구조
| 층 | 설명 | 수량 | 예시 |
|----|------|------|------|
| 니치 태그 | 구체적 주제 (경쟁 낮음, 타겟 정확) | 5~7개 | #헤어컬러변신 #미용사일상 #살롱케어 |
| 트렌드 태그 | 현재 유행 키워드 (노출 확대) | 3~5개 | #2026헤어트렌드 #여름헤어 |
| 브랜드/일반 태그 | 대형 카테고리 (발견성) | 2~3개 | #뷰티 #헤어 #미용 |

### 플랫폼별 해시태그 차이
| 플랫폼 | 최적 수량 | 특징 |
|--------|----------|------|
| 인스타 | 10~15개 | 니치+트렌드 혼합, 릴스는 5~8개면 충분 |
| 틱톡 | 3~5개 | 적을수록 좋음, 트렌드 태그 필수 |
| 유튜브 쇼츠 | 3~5개 (설명란) | 제목 키워드가 더 중요 |

### 미용사 필수 해시태그 풀
- 직업: #미용사 #헤어디자이너 #스타일리스트 #미용실
- 기술: #헤어컬러 #펌 #매직 #커트 #탈색 #염색
- 콘텐츠: #비포애프터 #헤어변신 #시술과정 #헤어꿀팁
- 타겟: #셀프헤어 #홈케어 #헤어관리 #두피케어

## 업로드 시간대 (한국 기준)

### 플랫폼별 최적 업로드 시간
| 플랫폼 | 최적 시간 | 차선 시간 | 피해야 할 시간 |
|--------|----------|----------|-------------|
| 인스타 릴스 | 오전 7~9시, 오후 6~8시 | 점심 12~1시 | 새벽 1~5시 |
| 유튜브 쇼츠 | 오후 5~7시 | 오전 10~12시 | 새벽 2~6시 |
| 틱톡 | 오후 7~10시 | 오전 8~10시 | 오전 3~6시 |
| 유튜브 롱폼 | 오후 2~4시 (주중), 오전 10~12시 (주말) | 오후 6~8시 | 월요일 오전 |

### 요일별 최적
- 화~목: 가장 높은 노출 (주중 활동 피크)
- 일요일: 롱폼 시청 시간 가장 김
- 월요일: 가장 낮은 참여율 (피해야 함)
- 금/토: 중간 (주말 시작 분위기)

## 알고리즘 기본 (플랫폼별 핵심 지표)

### 유튜브 쇼츠
- 핵심 지표: 스와이프율 (넘기지 않고 본 비율)
- 2차 지표: 좋아요, 댓글, 구독 전환
- 전략: 첫 2초에 시선 고정, 루프 구조 (끝→처음 자연스럽게)
- 주의: 워터마크(다른 플랫폼 로고) 포함 시 노출 페널티

### 인스타그램 릴스
- 핵심 지표: 완주율 (끝까지 본 비율)
- 2차 지표: 저장, 공유, 릴스 재시청
- 전략: 15~30초가 완주율 가장 높음, 자막 필수 (음소거 시청 고려)
- 주의: 저화질 영상 노출 제한, 최소 1080x1920

### 틱톡
- 핵심 지표: 시청 완료율 + 반복 시청
- 2차 지표: 공유, 댓글, 좋아요
- 전략: 15~30초 최적, "루프" 구조 강력 추천
- 주의: 다른 플랫폼 워터마크 절대 금지

## 에이전트 활용 가이드

이 파일을 참조할 때 에이전트는:
1. 영상 아이디어 작성 시 훅 공식 5개 중 적합한 것을 선택/제안
2. 각 아이디어에 추천 구조(변신형/교육형/실패수정형) 매칭
3. 제목 패턴표를 활용하여 실제 올릴 수 있는 제목 형태로 작성
4. 해시태그는 3층 구조로 구체적으로 추천
5. 업로드 시간대는 플랫폼 추천 결과와 연동하여 제안
