# 미용사 콘텐츠 아이디어 뱅크
# 목적: 컨셉기획 에이전트가 미용사 대상 구체적 영상 아이디어를 도출할 때 참조
updated: 2026-05-21

---

## 핵심 원칙

아이디어 뱅크는 "그대로 쓰는 것"이 아니라 대상자에 맞게 변형하는 소스다.
에이전트는 대상자의 강점/성격/타겟에 맞는 아이디어를 선택/조합/변형한다.
모든 아이디어에는 난이도와 추천 포맷이 명시되어 있다.

난이도 기준:
- 초급: 촬영 경험 없어도 가능, 편집 최소, 얼굴 노출 불필요
- 중급: 기본 편집 필요, 음성 또는 부분 얼굴 노출
- 고급: 복잡한 편집, 스토리 구성, 얼굴 노출 필요

| 1-2 | 펌 전체 과정 타임랩스 | 쇼츠 30~60초 | 초급 | 꾸준한 조회 |
| 1-3 | 손상모 복구 과정 (before → 시술 → after) | 쇼츠 30초 | 초급 | 높은 바이럴 확률 |

| 1-5 | 남성 헤어 변신 (장발 → 숏컷 등) | 쇼츠 30초 | 초급 | 남성 타겟 확보 |
| 1-6 | 탈색 과정 실시간 (다회 탈색) | 롱폼 10~15분 | 중급 | 마니아 시청 |

### 활용 팁
- 1-1, 1-3: 내향적 크리에이터에게 최적 (손만, 자막만으로 가능)
- 1-4: 댓글 참여율 높음 → 알고리즘에 긍정 신호
- 1-6: 전문성 과시 + 긴 시청 시간 → 유튜브 수익화에 유리

## 시리즈 콘텐츠 추천 (구독 전환에 효과적)

### 추천 시리즈

| 시리즈명 | 콘셉트 | 빈도 | 적합 성격 |
|---------|--------|------|----------|
| "수습 성장기" | 경력 성장 과정 기록 | 격주 | 초보 미용사, 감성형 |
| "이 주의 변신" | 매주 베스트 시술 1개 | 주 1회 | 기술 자신감 있는 미용사 |
| "미용사 꿀팁" | 짧은 팁 시리즈 (번호 매기기) | 주 2~3회 | 교육형, 논리형 |
| "고객 사연" | 고객의 헤어 고민 + 해결 | 격주 | 소통형, 감성형 |
| "제품 솔직 리뷰" | 미용사 관점 제품 평가 | 월 2~4회 | 논리형, 협찬 목표 |
| "미용 상식 퀴즈" | 짧은 퀴즈 형태 숏폼 | 주 1~2회 | 재밌는 성격, 외향형 |

### 시리즈의 장점
- 구독 전환: "다음 편도 보고 싶다" → 구독
- 제작 루틴: 매번 새 기획 불필요 → 지속 가능성 높음
- 알고리즘: 같은 시리즈 영상끼리 추천 연결

## 콘텐츠 난이도별 추천 순서 (초보 크리에이터)

### Phase 1: 첫 2주 (난이도: 초급만)
- 1-1 컬러 변신 쇼츠 x 3
- 1-3 손상모 복구 쇼츠 x 2
- 2-7 손상 등급 자가진단 x 1
- 3-6 미용실 청소 루틴 x 1
총 7개. 목표: 촬영-편집 루틴 확립

### Phase 2: 3~4주 (초급 + 중급)
- 2-1 드라이 방법 교육
- 2-5 샴푸 고르는 법
- 3-1 미용사 하루
- 4-2 셀프 염색 실패 복구
총 4개 + Phase 1 반복. 목표: 콘텐츠 다양화 시작

### Phase 3: 5~8주 (중급 중심)
- 시리즈 1개 시작 (위 추천에서 성격 매칭)
- 5-1, 5-2 Q&A형 도전
- 2-3, 2-6 롱폼 교육 콘텐츠
목표: 채널 정체성 확립

## 에이전트 활용 가이드

이 파일을 참조할 때 에이전트는:
1. 대상자의 성격/경력/카메라 경험에 따라 적합한 카테고리 우선 선택
2. 난이도를 대상자 수준에 맞게 필터링
3. 아이디어를 그대로 쓰지 않고 대상자 특기/타겟에 맞게 변형
4. 5개 아이디어 도출 시 카테고리 2개 이상에서 선택 (다양성)
5. 시리즈 추천은 장기 전략 섹션에서 활용
6. 난이도별 순서는 실행 가이드/캘린더 작성 시 참조
