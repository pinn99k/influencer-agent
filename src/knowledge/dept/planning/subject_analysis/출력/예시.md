# 대상분석 좋은/나쁜 예시
# 목적: 강점·차별점 작성 품질 기준. 복사 금지 — 패턴만 학습한다.
updated: 2026-05-29

---

입력 예: 이름=김민수, 직업=미용사 수습생(1년차), 특기=헤어컬러링/블리치,
성격=내향적·섬세, 타겟=20대 여성

## 강점
GOOD: "블리치·컬러링 시술의 단계별 실패 포인트를 살롱 현장에서 직접 보여줄 수 있다 — 콘텐츠 형식: 시술 과정 숏폼 — 근거: 직업, 특기"

BAD: "초기 플랫폼 사용 경험" (팔로워 수는 현황이지 강점이 아님)
BAD: "미용 전문 지식이 있다" (콘텐츠 형식이 없음 → generic)

## 차별점
GOOD: "헤어 컬러링 수습생의 섬세한 시각으로 20대 여성에게 '블리치 전에 반드시 알아야 할 것들'을 제공한다."

BAD: "높은 수준의 미용 콘텐츠로 20대 여성에게 큰 매력으로 작용할 수 있습니다." (막연한 가능성 표현)
