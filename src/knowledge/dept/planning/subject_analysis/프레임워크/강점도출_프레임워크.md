# 강점 도출 프레임워크 (직업 범용)
# 목적: 대상분석 에이전트가 어떤 직업의 크리에이터든 강점을 콘텐츠 자산으로 전환할 때 참조
updated: 2026-05-29

---

## 핵심 명제
강점은 "이 사람이기에 만들 수 있는 콘텐츠"다. 능력 자체가 아니라, 그 능력이 만들어내는
콘텐츠 형식까지가 강점이다. "전문성이 있다"는 강점이 아니다. "전문성으로 어떤 형식의
콘텐츠를 만든다"가 강점이다.

---

## 4대 강점 원천
입력 정보를 아래 4개 원천으로 분해해 강점 후보를 도출한다.

### 1. 직업 — 내부자 시점
직업은 외부인이 모르는 것을 아는 희소성의 원천이다.

| 직업군 | 내부자 자산 | 가능한 콘텐츠 형식 |
|--------|-----------|------------------|
| 전문직(의사·변호사·회계사) | 일반인이 오해하는 지식, 판단 기준 | 오해 정정, 사례 해설, 의사결정 공개 |
| 서비스직(미용사·트레이너·바리스타) | 현장 노하우, 고객 패턴 | 작업 과정, 고객 유형별 팁, before/after |
| 기술직(개발자·엔지니어·정비사) | 문제 해결 과정, 도구 사용법 | 과정 타임랩스, 트러블슈팅, 도구 리뷰 |
| 창작직(디자이너·작가·요리사) | 창작 과정, 취향 기준 | 메이킹, 비포애프터, 큐레이션 |
| 일상·취미 기반 | 몰입 경험, 시행착오 | 입문 가이드, 실패담, 성장 기록 |

도출 질문: "이 직업의 사람이 식사 자리에서 말하면 다들 신기해하는 얘기는 무엇인가?"

### 2. 특기 — 복제 난이도
특기는 남이 따라 하기 어려운 정도로 평가한다.
- 높은 복제난이도(수년 숙련 필요): 핵심 강점. 과정·결과를 보여주는 것만으로 콘텐츠가 된다.
- 낮은 복제난이도(누구나 가능): 단독으로는 강점 아님. 다른 원천과 결합해야 의미.

도출 질문: "이 특기로 만든 결과물을 보면 사람들이 '나는 저렇게 못 해'라고 느끼는가?"

### 3. 성격 — 콘텐츠 스타일 전환
성격은 약점이 아니라 스타일이다. 콘텐츠 포맷으로 전환한다.

| 성격 | 전환된 콘텐츠 스타일 |
|------|-------------------|
| 내향적·관찰형 | 말하기보다 보여주기. 과정 영상, 자막 해설, 잔잔한 포맷 |
| 외향적·소통형 | 말하기 중심. 질문답변, 라이브, 리액션, 토크 |
| 분석적·꼼꼼형 | 비교·검증·랭킹·데이터 콘텐츠 |
| 감성적·공감형 | 스토리텔링, 일상 기록, 위로·동기부여 |

도출 질문: "이 성격이 카메라 앞에서 부담인가 무기인가? 무기가 되는 포맷은?"

### 4. SNS 경험 — 출발 자산 (강점이 될 때만)
주의: 팔로워 수·게시물 수는 현황이지 강점이 아니다.
경험이 강점이 되는 경우: 편집 숙련도, 특정 플랫폼 알고리즘 이해, 기존 커뮤니티 보유.
해당 없으면 강점에서 제외하고 약점·출발점으로 다룬다.

---

## 강점 작성 기준
- 각 강점은 [능력] + [콘텐츠 형식] + [근거 필드] 3요소를 갖춘다.
- 3개로 압축한다. 많을수록 집중이 흐려진다.
- 강점끼리 겹치면 묶고, 가장 날카로운 3개만 남긴다.

## 흔한 실수
- "다양한 콘텐츠 제작 가능": 어떤 형식인지 없음 = 강점 아님
- "전문 지식 보유": 콘텐츠 연결 없음 = 강점 아님
- 팔로워 수를 강점으로 기재: 금지
